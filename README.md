## Real Time Sports Data
Our project will be a public backend API for a sports betting simulator allowing users to wagers on real world games using virtual currency. Inspired by platforms like ESPN and DraftKings our API combines live game data with a fully interactive betting. Users can browse live games, place bets, and track their performance over time. while the API manages user balances, bet history, and outcomes. When games conclude, the system automatically resolves bets and updates standings based on results. The platform will include leaderboards and comments to encourage competition among users. We will be using live data and a persistant relational database to store all data.
